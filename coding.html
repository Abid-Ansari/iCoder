Geeks for Geeks
The most famous website among tech learners

It is the most dynamic website suitable for all range of students from beginner to advanced.It is most widely used website by students as it has all the resources to slove anyone's doubt. Almost every programming language tutorials are available on this website like C,C++,Java,Python etc.Apart from this almost every computer science fundamental subjects like OOPS, DBMS, Networking and many more are available for students. All the elements of web designing and tools used are explained with great details with a lot of projects.One can learn HTML,CSS,JavaScript,Bootstrap and many more things in a very lucid way.In one line, it can be defined as an encyclopedia for anyone who is interested in computer science domain.

It is one of the best platform to grasp the concepts of data structures and algorithms and develop problem solving attitude. It is also beneficial in the interview preparation journey of a student.

LeetCode
Crisp the code

Anybody job hunting for a software development position is likely to know Leetcode very well, the website collects questions and solutions that come up in technical interviews for some of the most sought-after companies like Google, Facebook, and Microsoft. These kinds of interview problems involving data structures and algorithms are becoming more and more popular even amongst smaller companies and this is a cause of frustration for many that argue their “leetcoding” skills don’t reflect their ability to do the job.
